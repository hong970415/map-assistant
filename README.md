# map-assistant

All logic and code are written in views/HomeView.vue

## Getting Started

### Prerequisites

```bash
node -v >= 18
yarn -v >= 1.22
```

### Installation

```bash
yarn install
```

### To Start Development Server

1. create .env file and enter API Key value

```bash
VITE_GOOGLE_MAP_API_KEY=
```

2. start

```bash
yarn dev
```

3. Visit http://localhost:3000
